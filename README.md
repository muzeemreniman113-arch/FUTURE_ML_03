Customer Support Chatbot – Task 3
Internship
Future Interns – Machine Learning Internship (Track Code: ML)

Task Description
This project implements a customer support chatbot that answers frequently asked questions using NLP-based keyword matching.

Features
- Handles customer support queries
- FAQ-based responses
- Smart fallback handling
- Web-based chatbot interface
- Deployed using Streamlit

Tools & Technologies
- Python
- Streamlit
- NLP (Keyword Matching)

How to Run the Project
1. Install dependencies:
   pip install -r requirements.txt

2. Run the chatbot:
   streamlit run app.py

Output
A fully functional customer support chatbot accessible via a web browser.
